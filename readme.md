    - - [x] Forms and Validations
    - - [ ] Writing Semantic HTML (optional)
    - - [ ] SEO Basic (optional)
    - - [ ] Accessibility (optional)
  - CSS
    - - [x] Learn the basics
    - - [x] Making Layouts

        - - [x] Positioning

    - - [x] Learn DOM Manipulation
