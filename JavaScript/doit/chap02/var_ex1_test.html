변수에 저장할 수 있는 자료형은 문자형(String), 숫자형(Number), 논리형(Boolean) 그리고 빈 데이터(Undefined)
        1. 문자형
        - var 변수 = "사용할 문자나 숫자";
        var s = "javascript";
        var num = "100";
        var tag = "
String
";

        2. 숫자형
        - var 변수 = 숫자; 또는 Number("문자형 숫자");
        var s = 100;
        var t = Number("500");  // "500" -> 500

        3. 논리형
        - var 변수 = true or false; 또는 Boolean(데이터);
        var s = true;
        var t = 10 >= 100 // false
        var k = Boolean("hello") // true
        - Boolean()메서드는 숫자0과 null, undefined, 빈문자("")를 제외한 모든 데이터에 대해 true를 반환합니다.
